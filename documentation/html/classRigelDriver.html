Detailed Description

This is a library for manouvering a Budget Robotics Rigel robot car.

The constants are set for the servo configuration where the wheels are set as far apart as possible. The Adruino digital pins are also defined in the constants header file. This class most likely has very little value to anyone but me since this is highly specific to my configuration. This file is included mainly for completeness. Depends on SoftwareServo.
