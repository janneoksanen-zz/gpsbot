Detailed Description

Navigator object for a GPS robot car.

I wrote this lib for my GPS navigating robot car project. It was meant to be used with my Waypoint and GarminParser classes and some of the design peculiarities are inherited from those classes. This class mainly calculates distance and bearing to destination with the information provided.
